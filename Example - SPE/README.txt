To use the Strileckis Practice Environment...


1. Run the PracticeEnvironment.jar file
2. Enter a name that you would like to go by and press the "Submit" button
3. When the environment loads, press the "Load new lesson" button
4. Choose the .spef file in the same folder (note: you can move these to separate folders without issue)
5. Complete the activity by copying what is in the left side, into the right side
6. At times, you will need to fill in a blank
7. After completing the activity, there will be a small, encrypted report file with your name on it
8. (If you were in my class) Submit that report to me for credit
